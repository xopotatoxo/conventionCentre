EventEASE Vendor Details Page Guide

Welcome to the EventEASE Vendor Details page. This guide will assist you in navigating and using the features available on this page.

  • Vendor Information: The page displays specific details about the selected vendor, including their name and a brief description. Note that the current information is a placeholder and the page is under construction.
  • Interacting with Vendor Options: Click the 'Map' button to view the vendor's location within the event venue. Use the 'Check-In' button to mark your visit to the vendor. This button toggles to 'Checked-In' once clicked. The 'Notes' button opens a modal where you can jot down any notes or reminders related to the vendor.

As always, the navbar is present along the bottom of the screen. Use the home icon to return to the main menu and navigate elsewhere. The question mark icon opens this help dialog.
